## 摘要
多模态大语言模型（MLLMs）在视觉理解方面取得了显著成功，但由于语义感知与像素级重建之间的特征差异，在视觉生成方面仍受限制。弥合这一差距需要克服两个核心挑战：赋予语义编码器高保真重建能力，并在不依赖外部教师的情况下将生成模型与语义空间有效对齐。为此，我们提出了一种新颖的统一多模态框架，其特点是语义-像素自对齐与自适应路由（SPAR）。首先，为了协调语义感知与像素级重建，我们引入了一种非对称双流统一分词器。轻量级语义流锚定判别性特征，而Transformer增强的像素流将细粒度视觉细节恢复到统一的紧凑潜在空间中。其次，为了消除外部依赖，我们提出了一种自对齐生成范式，该范式原生地利用这一优化后的分词器作为扩散模型的内部对齐教师。此外，为了在统一的潜在空间中促进灵活的多模态交互，我们引入了动态令牌路由，使每个令牌能够根据其不同的语义需求自适应地聚合多层MLLM特征。大量实验表明，SPAR为统一架构建立了最新技术水平，在保持基础视觉理解能力的同时，实现了卓越的生成和重建质量。

## Abstract
Multimodal Large Language Models (MLLMs) have achieved remarkable success in visual understanding but remain constrained in visual generation due to the fundamental feature discrepancy between semantic perception and pixel-level reconstruction. Bridging this gap requires overcoming two core challenges: endowing semantic encoders with high-fidelity reconstruction capabilities, and effectively aligning generative models with semantic spaces without relying on external teachers. To this end, we propose a novel unified multimodal framework featuring \textbf{S}emantic-\textbf{P}ixel self-alignment and \textbf{A}daptive \textbf{R}outing (\textbf{SPAR}). First, to reconcile semantic perception with pixel-level reconstruction, we introduce an asymmetric dual-stream unified tokenizer. A lightweight semantic stream anchors discriminative features, while a Transformer-augmented pixel stream recovers fine-grained visual details into a unified compact latent space. Second, to eliminate external dependencies, we propose a self-aligned generation paradigm that natively leverages this optimized tokenizer as an internal alignment teacher for the diffusion model. Furthermore, to facilitate flexible multimodal interaction within this unified space, we introduce Dynamic Token Routing, which enables each token to adaptively aggregate multi-layer MLLM features based on its distinct semantic demands. Extensive experiments demonstrate that SPAR establishes the state-of-the-art for unified architectures, achieving exceptional generation and reconstruction quality while preserving foundational visual understanding capabilities.

## 2. 方法论：核心思想与关键技术

SPAR 框架由三个关键部分组成：**语义-像素双流自对齐统一分词器**、**动态令牌路由**以及**自对齐生成范式**，三者协同解决特征鸿沟与外部依赖问题。

### 2.1 非对称双流统一分词器

输入图像经冻结的语义编码器 \(E_s\) 得到高维特征 \(f_v\)，后分两路处理：

- **语义流（轻量锚点）**：采用残差块 + MLP 投影，映射至紧凑潜在空间：
  \[
  z_s = P_s(f_v) \in \mathbb{R}^{H' \times W' \times D_p}
  \]
  该流严格锚定原语义特征，防止灾难性遗忘。
- **像素流（重建桥梁）**：配备独立的 Transformer 编码器 \(T_p\)（建模长程依赖），将语义特征重构为适合像素解码的中间表示：
  \[
  f_p = T_p(f_v),\quad z_p = P_p(f_p) \in \mathbb{R}^{H' \times W' \times D_p}
  \]
  随后 \(z_s\) 与 \(z_p\) 在通道维拼接，经融合层 \(F\) 映射回潜在空间：
  \[
  z_f = F\big([z_s; z_p]\big)
  \]
  最终由预训练像素解码器 \(D_p\) 重建图像。该设计明确将语义保留与像素重建解耦。

**渐进三阶段训练**：

- **阶段一（双流初始化）**：冻结编码器与解码器，仅训练双流模块。引入像素对齐损失 \(L_p = \|z_p - z'_p\|_2^2\)，其中 \(z'_p\) 由冻结的像素编码器输出，引导像素流进入解码器的原生潜在流形。
- **阶段二（联合解码器训练）**：解冻解码器，仅用重建损失联合优化。
- **阶段三（编码器微调与自蒸馏）**：解冻编码器，同时引入语义对齐损失 \(L_s = \|f_v - f'_v\|_2^2\)，用冻结编码器的特征 \(f'_v\) 限制当前特征，配合低学习率和 GAN 判别损失，实现无遗忘的像素能力增强。

### 2.2 统一多模态模型与动态令牌路由

- **条件信号构建**：在输入序列中预留位置插入可学习查询嵌入，经 MLLM 因果自注意力聚合上下文，形成融合多模态语义的条件表示。
- **动态令牌路由（DTR）**：从 MLLM 的 \(L+1\) 层隐藏态中等间隔采样 \(K\) 层，用最顶层特征 \(H^{(K)}\) 作为查询，通过轻量路由网络 \(g\) 计算每个令牌自适应的层融合权重 \(w_i = \sigma(g(H_i^{(K)})/\tau)\)，再与可学习缩放因子 \(\alpha\) 加权求和并用线性投影得到最终表示。该机制使边缘、纹理、指令等不同语义角色的令牌能动态利用不同层级的特征。
- **自对齐生成范式**：直接用训练好的双流分词器作为内部对齐教师。提取扩散模型（DiT）中间层特征 \(f_{dit}\)，经投影头 \(\phi_a\) 映射后，与分词器编码器投影特征 \(f'_v\) 计算余弦相似度损失：
  \[
  L_{align} = -\frac{1}{N}\sum_i \cos(\phi_a(f_{dit,i}), f'_{v,i})
  \]
  避免引入任何外部预训练特征网络。

**三阶段统一训练**：  
1. 连接器预训练（冻结 MLLM 和 DiT，训练连接器、查询嵌入、DTR）。  
2. 联合预训练（解冻 DiT，加入流匹配损失与自对齐损失）。  
3. 监督微调（使用高质量指令数据进一步优化）。

## 6. 主要结论与发现

- 非对称双流设计成功**解耦**了语义保留与像素重建：语义流作为锚点防止遗忘，像素流显著提升重建细节（PSNR 提升 2+ dB）。
- 渐进训练配合语义对齐损失，使编码器在获取像素能力的同时**未损失甚至略微增强**原有理解性能。
- 自对齐生成范式**彻底消除了对外部教师的依赖**，同时利用分词器内部知识有效提升生成质量。
- 动态令牌路由通过自适应层融合进一步挖掘 MLLM 层次表征潜力，带来编辑与生成指标的全面增益。
- SPAR 在统一架构下实现了 **SOTA 的重建质量**、**领先的理解表现**以及**极具竞争力的生成与编辑效果**（如 GenEval 3B 达到 0.91，ImgEdit 4.01），证明统一空间完全可以同时支撑两类任务。

## 7. 方法亮点与优势

- **解耦而非折衷**：通过不对称双流结构，避免了以往方法为迁就重建而损害理解的弊端。
- **无外部教师的自对齐**：利用自身分词器做对齐，使生成与理解共享一致的语义空间，消除了异源特征导致的偏差。
- **可解释的动态路由**：DTR 给出令牌级别的层偏好分布，为理解模型内部机制提供了可视化窗口。
- **训练策略完善**：三阶段分词器训练与三阶段统一模型训练设计严谨，各阶段损失函数明确，循序渐进。
- **实验全面且具说服力**：在重建、理解、生成、编辑四大维度上与大量近期基线对比，并通过消融验证了每个关键组件的必要性。
