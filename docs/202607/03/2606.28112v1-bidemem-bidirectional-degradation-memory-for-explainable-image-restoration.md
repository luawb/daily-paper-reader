## 摘要
基于退化感知的提示、条件和潜在先验越来越多地被用于图像复原，但它们通常仅通过单一终点来评判：复原图像是否获得更高的PSNR。这是对语义的弱检验。条件可以通过增加容量、充当全局校正偏差或利用数据集捷径来提供帮助，而无需成为可解释的退化先验。我们提出BiDeMem，一种用于可解释图像复原的双向退化记忆。由复原特征和输入统计构建的查询检索出紧凑的top-k记忆槽子集。所选的相同槽身份在推理时支持复原路径，并在仅训练时支持前向退化解释路径。研究集中在受控多退化NAFNet设置中的可验证性。新的控制措施将增益单独分离为校正头部、密集查询先验和静态全局先验：这些变体分别比BiRank低0.2588、0.2586和0.2839 dB。强残差监督和更宽的退化头部也仍低于完整的双向记忆模型。干预探测表明，BiRank在保持复原质量的同时增加了错误先验和原生先验的敏感性，将退化记忆构建为既是复原模块又是可证伪的解释机制。

## Abstract
Degradation-aware prompts, conditions, and latent priors are increasingly used in image restoration, yet they are usually judged by a single endpoint: whether the restored image obtains higher PSNR. This is a weak test of semantics. A condition can help by adding capacity, acting as a global correction bias, or exploiting dataset shortcuts, without becoming an interpretable degradation prior. We propose BiDeMem, a bidirectional degradation memory for explainable image restoration. A query built from restoration features and input statistics retrieves a compact top-k subset of memory slots. The same selected slot identity supports the restoration path at inference time and a training-only forward-degradation explanation path. The study centers on verifiability in a controlled multi-degradation NAFNet setting. New controls separate the gain from a correction head alone, a dense query prior, and a static global prior: these variants are 0.2588, 0.2586, and 0.2839 dB below BiRank, respectively. Strong residual supervision and a wider degradation head also remain below the full bidirectional memory model. Intervention probes show that BiRank preserves restoration quality while increasing wrong-prior and native-prior sensitivity, framing degradation memory as both a restoration module and a falsifiable explanation mechanism.

### 主要结论与发现
- **受控复原**：BiRank Memory（双向）在 8 个已见基准上平均 PSNR 29.75 dB，优于 NAFNet、NAFNet‑wide 和 Rank Memory，但优势微小，核心价值不在绝对增益。
- **控制消融**：校正头单独、密集查询先验、静态全局先验分别比 BiRank 低 0.26 dB 左右，说明记忆检索通路而非额外容量或简单条件带来了增益。
- **双向干预敏感**：与 Rank Memory 相比，BiRank Memory 的错误先验下降从 0.24 dB 增至 1.04 dB，原生/非原生差距从 0.35 dB 增至 0.61 dB，打乱和活动槽屏蔽效应也更显著，证明双向设计强化了先验的因果依赖性和可检验性。
- **泛化与复用**：未见/混合退化下记忆变体持续优于基线；低数据微调（10%~50% 数据）表现出大幅增益，表明紧凑记忆具有良好的可复用性。
- **外部骨干验证**：在 AirNet 和 PromptIR 上，BiRank Memory 均获得最佳 PSNR，但 SSIM 不总是最优，效果依赖于具体设置。

### 优点
- **可解释性导向**：将退化先验从黑箱条件提升为可审计对象，通过双向约束和多维干预实验进行验证，这在图像复原领域较为新颖。
- **严谨的消融设计**：通过一系列控制变体（容量控制、校正头控制、密集先验控制、静态先验控制等）逐一排除替代解释，使结论更可靠。
- **双向一致性**：训练时要求相同记忆槽既能辅助复原，又能解释退化过程，引入内在的语义约束。
- **紧凑且可复用**：记忆规模小（12 槽，4 活动），在低数据场景下表现突出，便于迁移和适应。
- **广泛的干预评估**：不仅报告平均 PSNR，还系统性地评测错误先验、打乱、屏蔽、原生/非原生对比等多种反事实干预的效果。

### 不足与局限
- **计算效率不高**：记忆与校正头增加了延迟（256×256 输入下约 42 ms），参数虽少但速度未提优。
- **架构限制**：基于紧凑全局槽的设计未必在所有骨干网络或退化混合上最优，适用性有待进一步验证。
- **外部验证尚不充分**：迁移到 AirNet、PromptIR 时增益依赖于训练方式和指标，SSIM 并不总能提升，外部证据仍属“支持性但不完全确定”。
- **未详细报告算力与时长**：缺少 GPU 型号、数量、训练时间等信息，可复现性受一定影响。
- **双向分支仅在训练时使用**：推理时仍为单向复原，但其带来的解释性收益需要在实际部署中与计算成本权衡。
