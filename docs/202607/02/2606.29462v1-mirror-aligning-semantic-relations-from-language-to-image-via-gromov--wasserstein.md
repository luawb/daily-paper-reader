## 摘要
多模态大语言模型（MLLMs）从其语言骨干中继承了丰富的关系先验，但在要求将这些关系应用于视觉情境时往往失败。我们将此失败追溯至一个结构性的盲点：基于投影的对齐训练每个视觉标记携带正确的语义，却从未询问概念之间的关系是否在从语言到视觉的跨越中得以保留。为解决此问题，我们提出了MIRROR（通过基于最优传输的正则化将概念间关系从语言映射到视觉表示），这是一个几何正则化框架，通过利用语言表示中编码的丰富关系结构，将关系先验从语言迁移到视觉。具体而言，我们从所提出的半逆Gromov-Wasserstein（SI-GW）问题中导出一个替代损失，该逆几何问题将视觉表示与源自语言的关系先验对齐。我们证明此公式允许一个唯一的闭式解，该解规定了由语言几何和跨模态耦合所暗示的理想视觉关系结构。该公式的结构还支持高效计算，使其适用于长标记序列。在仅解码器Transformer内部应用SI-GW需要精心设计。我们引入了层、头和标记级别的针对性策略，以确保无需额外参数或推理成本即可稳定提取。MIRROR在保持通用视觉-语言任务性能的同时，提高了关系一致性。

## Abstract
Multimodal Large Language Models (MLLMs) inherit rich relational priors from their language backbones, yet often fail when asked to apply these relationships in visual contexts. We trace this failure to a structural blind spot: projection-based alignment trains each visual token to carry the right semantics, but never asks whether the relationships between concepts survive the crossing from language to vision. To address this, we propose MIRROR (Mapping Inter-concept Relations from language to visual Representation via Optimal-transport-based Regularization), a geometric regularization framework that transfers relational priors from language to vision by exploiting the rich relational structure encoded in language representations. Specifically, we derive a surrogate loss from the proposed Semi-Inverse Gromov-Wasserstein (SI-GW) problem, an inverse geometric problem that aligns visual representations with language-derived relational priors. We show that this formulation admits a unique closed-form solution that prescribes the ideal visual relational structure implied by language geometry and cross-modal coupling. The structure of the formulation also enables efficient computation, making it applicable to long token sequences. Applying SI-GW inside decoder-only Transformers requires careful design. We introduce targeted strategies at the layer, head, and token levels to ensure stable extraction without additional parameters or inference cost. MIRROR improves relational consistency while preserving performance on general vision-language tasks.

### 1. 论文的核心问题与整体含义（研究动机和背景）
-   **核心问题**：多模态大语言模型（MLLMs）虽然在文本推理中能利用来自语言模型的关系先验（如空间配置、逻辑依赖），但当被要求将这些关系知识应用于视觉理解时，往往会出现“语义关系不传递”的问题——语言上知道“坐着要求重心在支撑面之上”，视觉上却判定错误。
-   **根因分析**：主流 MLLMs 通过投影适配器将视觉 token 对齐到语言嵌入空间，并使用下一 token 预测目标训练。这种训练范式仅确保每个视觉 token **独立地**携带正确的语义（身份级对齐），但从未显式监督概念间的**成对关系结构**是否在跨模态过程中得到保留，造成了“结构盲点”。
-   **整体含义**：有效跨模态理解不仅需要对齐“是什么”，还需要对齐“彼此如何关联”，即需要将语言空间内编码良好的关系几何结构（距离/相似度）迁移并反映到视觉表示中。

### 2. 论文提出的方法论：核心思想、关键技术细节
-   **核心思想**：将跨模态关系对齐建模为 **半逆 Gromov–Wasserstein（SI-GW）问题**，以语言 token 间的距离矩阵 \(D_t\) 和文本到视觉的交叉注意力耦合矩阵 \(C\) 为固定输入，求出能使两种模态在关系上相一致的**理想视觉距离矩阵 \(\hat{D}_v\)**，并以此作为正则化目标驱动视觉几何重构。
-   **关键技术细节**：
    -   **几何提取**：利用 Transformer 自注意力图作为语义关系度量，经对称化、对数变换得到文本距离 \(D_t\) 和视觉距离 \(D_v\)。
    -   **耦合矩阵**：直接从 LLM 中间层的文本到视觉交叉注意力获取软分配矩阵 \(C\)。
    -   **半逆 GW 问题与闭式解**：给定 \(D_t\) 和 \(C\)，求解使 GW 目标最小的视觉几何 \(\hat{D}_v\)，推导出唯一闭式解：
        \[
        \hat{D}_v = (C^\top D_t C) \oslash (bb^\top)
        \]
        其中 \(b = C^\top\mathbf{1}\)。该解直观理解为：视觉 token 对 \((k,\ell)\) 的理想距离，等于所有关注它们的文本 token 对之间距离的期望。
    -   **SI-GW 损失**：定义为当前 \(D_v\) 与理想 \(\hat{D}_v\) 的 Frobenius 范数平方差，即 \(L_{\text{SI-GW}} = \| D_v - \hat{D}_v \|_F^2\)。总损失 \(L_{\text{total}} = L_{\text{LM}} + \lambda L_{\text{SI-GW}}\)。
    -   **计算效率**：通过矩阵运算将复杂度从 \(O(n_t^2 n_v^2)\) 降至 \(O(n_t^2 n_v + n_v^2 n_t)\)，适合长 token 序列。
    -   **稳定化工程策略**（仅解码器 Transformer 内）：
        -   **层解耦**：\(D_t\) 取自早期 LLM 层（纯文本几何），\(D_v\) 取自最终 ViT 层，\(C\) 取自中间 LLM 层（避免联合归一化带来的梯度耦合）。
        -   **头部选择**：按交叉注意力熵值筛选 top-k 低熵头部，平均形成干净耦合。
        -   **非语义 token 过滤**：对系统提示、占位符等 token 施加位置感知权重衰减，净化 \(D_t\)。
    -   **无额外开销**：所有注意力矩阵均为前向过程已有，不引入新参数或推理成本。

-   **闭式解带来的实用价值**：通过矩阵运算即可计算目标几何，使训练额外开销极小，且避免手动调参或迭代求解器。

### 7. 优点
